# Phishdetect_ai
PhishDetect AI: ML-Based Detection of AI-Generated Phishing Emails 🛡️  Final year project that classifies emails into Legitimate, Traditional Phishing, and AI‑Generated Phishing using Logistic Regression and TF‑IDF. Features a Streamlit web app with LIME explainability, Legitimate, Traditional,  AI‑Phishing), and PDF report generation.
